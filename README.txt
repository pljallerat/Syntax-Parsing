TP de compilation : Analyse synthaxique et évaluation d'une expression arithmétique

************************************************************************************************************************

Vous pouvez compiler l'ensemble des fichiers avec la commande "make".
Celle-ci génère un exécutable "main"
Vous pouvez lancer l'éxecutable avec la commande "./main [param]". 
param est optionnel, il correspond au calcul à effectuer. Si param n'est pas présent, un calcul par défault est exécuté.
